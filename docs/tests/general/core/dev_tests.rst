Setup environment:
------------------

.. _m2k-emu-dev:

**M2k.Emu:**
    - Open Scopy.
    - Start the iio-emu process using "adalm2000" in the dropdown and 
      "ip:127.0.0.1" as URI.
    - Add the device in device browser.

.. _m2k-usb-dev:

**M2k.Usb:**
    - Open Scopy.
    - Connect an **ADALM2000** device to the system by USB.
    - Add the device in device browser.
