Data Logger Multiplot

Experimental Feature: The new multiplot functionality lets you create and organize several plots within the Data Logger, making it easy to compare and analyze data visually. You can move signals between plots and select channels or instruments to log, assigning each to the plot of your choice. This feature is experimental and must be enabled in Scopy Preferences under the Data Logger section.

How to Enable and Use

  1. Go to Preferences → Data Logger and enable the Multiplot feature (experimental).
  2. Open the Data Logger tool from the main menu.
  3. Create multiple plots as needed.
  4. Select channels or instruments to log, and assign them to different plots.
  5. Move signals between plots for optimal visualization.
  6. Use export features to save your multiplot data or images.

For more detailed information, please visit the official documentation.
