Package Manager

The new Package Manager in Scopy makes it easy to browse, install, update, and remove plugins and packages directly from the application. With a new user interface, you can install plugins with one click and manage dependencies seamlessly. Security is improved with package verification, and you can view detailed info or roll back to previous versions if needed.

How to Use

  1. Open the Package Manager from the main menu.
  2. Browse available packages or search for a specific plugin.
  3. Click Install or Update to manage your packages.
  4. Check the Installed tab to manage or remove existing packages.

For more detailed information, please visit the official documentation.
