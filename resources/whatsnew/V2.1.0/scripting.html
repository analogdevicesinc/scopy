Scripting Tool

The new scripting tool in Scopy lets you automate workflows and instrument control using built-in scripting capabilities. With the integrated script editor, you can write, edit, and run scripts directly in Scopy. Scripts have access to supported instruments, allowing you to perform measurements, automate data collection, and execute complex or repetitive tasks. This feature is experimental and must be enabled in the application settings.

How to Enable and Use

  1. Go to Preferences and enable the Scripting Tool (experimental feature).
  2. Open the Scripting tool from the main menu.
  3. Write or load a script in the editor.
  4. Click Run to execute the script and interact with your instruments.
  5. Review output and errors in the console panel.

For more detailed information, please visit the official documentation.
