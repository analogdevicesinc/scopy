IMU Analyzer Plugin

The IMU Analyzer plugin brings advanced Inertial Measurement Unit analysis to Scopy with stunning 3D visualization capabilities. This professional-grade tool provides real-time orientation analysis, multi-axis sensor monitoring, and interactive 3D displays for ADIS family IMU sensors.

Key Features

  • Real-time 3D Visualization: Interactive OpenGL-based 3D scene showing device orientation with smooth real-time updates
  • ADIS Family Support: Compatible with ADIS16550, ADIS16545, ADIS16547, ADIS16505, and other ADIS IMU sensors
  • Multi-Axis Monitoring: Comprehensive 6-DOF sensing with 3-axis gyroscope and accelerometer data
  • Professional Interface: Tabbed interface with measurement mode, interactive controls, and detailed data panels
  • Advanced Data Processing: Delta angle and velocity calculations, temperature compensation, and filtering
  • Interactive Controls: Pan, zoom, rotate 3D view with mouse controls and one-click view reset

3D Visualization Engine

The plugin features a state-of-the-art 3D rendering engine that displays IMU orientation in real-time:

  • Scene Renderer: High-performance OpenGL-based 3D visualization
  • Real-time Updates: Smooth 60+ FPS orientation display with threaded data acquisition
  • Interactive Navigation: Mouse-based view manipulation with professional camera controls
  • Visual Clarity: Color-coded axes and clear orientation representation

Sensor Data Analysis

Comprehensive monitoring and analysis capabilities for all IMU sensor channels:

  • Angular Velocity: 3-axis gyroscope data with precision rad/s measurements
  • Linear Acceleration: 3-axis accelerometer with gravity compensation
  • Delta Measurements: Advanced delta angle and velocity calculations
  • Temperature Monitoring: Built-in temperature compensation for accuracy
  • Raw and Processed Data: Access to both raw sensor values and calculated parameters

Professional Tools

  • Measurement Mode: Toggle detailed numerical data panels with real-time updates
  • Calibration Support: Sensor bias adjustment and scale factor configuration
  • Filter Controls: Configurable low-pass filtering for noise reduction
  • High-Speed Sampling: Support for sampling rates up to 4000 Hz (ADIS16550)
  • Data Export: Logging and analysis export capabilities

ADIS Device Compatibility

The plugin automatically detects and supports ADIS family IMU sensors:

  • ADIS16550: High-performance tactical-grade IMU (primary target)
  • ADIS16545: Precision industrial IMU with advanced filtering
  • ADIS16547: Six degrees of freedom IMU for motion sensing
  • ADIS16505: Cost-effective MEMS IMU for general applications
  • Other ADIS Devices: Universal compatibility with ADIS family sensors

Advanced Architecture

Built with professional-grade architecture for reliable performance:

  • Threaded Processing: Separate threads for UI and data acquisition ensure smooth operation
  • OpenGL Acceleration: Hardware-accelerated 3D rendering for optimal performance
  • Signal/Slot Communication: Thread-safe data transfer between components
  • Modular Design: Separate components for rendering, data processing, and UI management

Requirements

  • Scopy v2.2.0 or later
  • ADIS family IMU sensor with IIO support
  • OpenGL-capable graphics hardware for 3D visualization
  • Multi-core processor recommended for optimal performance
