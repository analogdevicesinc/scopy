Genalyzer FFT Analysis

Scopy now integrates the Genalyzer library for advanced data converter testing and FFT analysis in the ADC plugin.

The Genalyzer integration provides comprehensive Fourier analysis capabilities, enabling detailed characterization of ADC performance metrics directly within the Frequency tool.

Analysis Modes

Auto Mode

In Auto mode, Genalyzer automatically detects the fundamental tone in the spectrum and computes analysis metrics. Configure the SSB width parameter to control the bin search range for tone detection.

Fixed Tone Mode

In Fixed Tone mode, specify a known signal frequency for precise analysis. This mode provides additional parameters:

  • Expected Frequency: The frequency of the signal tone in Hz
  • Harmonic Order: Maximum harmonic order for distortion analysis (HD2, HD3, etc.)
  • SSB Fundamental: Number of single-side bins for the fundamental tone
  • SSB Default: Default single-side bins for other components

Analysis Metrics

The Genalyzer analysis provides comprehensive ADC performance metrics including:

  • SNR: Signal-to-Noise Ratio
  • FSNR: Full-Scale-to-Noise Ratio
  • SINAD: Signal-to-Noise-and-Distortion Ratio
  • SFDR: Spurious-Free Dynamic Range
  • ABN: Average Bin Noise
  • NSD: Noise Spectral Density
  • Carrier/Harmonic levels: Individual tone magnitudes in dBFS and dBc

External Library

Note: The Genalyzer analysis feature requires the libgenalyzer library to be installed. This library is developed by Analog Devices for data converter testing and characterization. More information is available at: https://github.com/analogdevicesinc/genalyzer

How to Use

  1. Open the Frequency tool in the ADC plugin
  2. Enable Genalyzer analysis from the settings panel
  3. Select the analysis mode (Auto or Fixed Tone)
  4. Configure the analysis parameters as needed
  5. Start acquisition to see real-time analysis results

For more detailed information about Genalyzer metrics and analysis methodology, please visit the Genalyzer documentation.
