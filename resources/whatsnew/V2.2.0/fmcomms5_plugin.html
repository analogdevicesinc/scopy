FMCOMMS5 Plugin

Scopy now offers dedicated support for the FMCOMMS5 dual-channel RF transceiver evaluation board.

The new FMCOMMS5 Tool enables you to integrate and control FMCOMMS5-based devices within Scopy. It provides comprehensive features including dual AD9361 device configuration, multi-chip synchronization controls, and advanced calibration capabilities for precise multi-channel RF applications.

For advanced users, the FMCOMMS5 Advanced Tool offers access to low-level configuration and diagnostic features, including detailed gain control, BIST functionality, and expert-level parameter tuning for optimal performance.

Key Features

  • Dual AD9361 Support: Complete control of both AD9361 transceivers (ad9361-phy and ad9361-phy-B) with independent configuration
  • Multi-Chip Synchronization (MCS): Hardware synchronization controls for precise phase alignment between multiple channels
  • Real-time Configuration: All settings are immediately applied to hardware with automatic readback validation
  • Global Settings: ENSM modes, calibration controls, rate governors, and FIR filter configuration
  • Independent RX/TX Chains: Separate control for receive and transmit signal paths with tracking calibrations

How to Use

  1. Connect your FMCOMMS5 device to your computer.
  2. Open Scopy and select the FMCOMMS5 tool from the available instruments.
  3. Use the MCS Sync button to synchronize the dual AD9361 devices when needed.
  4. Access the FMCOMMS5 Advanced tool for low-level configuration and diagnostics.

For more detailed information, please visit the official documentation.
