PQM Plugin Updates

The PQM (Power Quality Monitor) Plugin receives significant improvements in Scopy v2.2.0, enhancing acquisition performance, UI flexibility, and data visualization capabilities for power quality analysis.

Key Features

  • Concurrent Acquisition: RMS and Harmonics instruments can now acquire data simultaneously. Waveform acquisition remains separate
  • Dockable Harmonics Widgets: The Harmonics instrument now features dockable widgets, allowing users to customize their workspace by arranging, resizing, and organizing UI components
  • Individual Current Channel Plots: By default, each current channel (Ia, Ib, Ic) has its own dedicated plot in the Harmonics view. An overview plot is also available for viewing voltage harmonics

How to Use

  1. Connect your PQM device to your computer.
  2. Open Scopy and select the PQM plugin from the available plugins.
  3. Start acquisition on RMS and Harmonics instruments simultaneously.
  4. In the Harmonics instrument, dock/undock widgets to customize your view.

For more detailed information, please visit the official documentation.
