RF Power Meter Plugin

Scopy now offers dedicated support for RF power measurement with the new RF Power Meter Plugin. This specialized tool transforms the Scopy Data Logger into a real-time RF power measurement instrument when connected to compatible hardware featuring the powrms IIO device.

Key Features

  • Forward Power Measurement: Real-time monitoring of forward RF power in dBm
  • Reverse Power Detection: Simultaneous measurement of reflected power in dBm for bidirectional monitoring
  • Corrected Voltage Readings: Temperature and calibration-corrected input/output voltage measurements in millivolts
  • Raw Voltage Monitoring: Uncorrected voltage measurements for diagnostic purposes
  • Intuitive Display: Seven-segment display interface for easy reading of measurements
  • Automatic Detection: Seamless device discovery when connecting compatible hardware

Hardware Requirements

  • ADI RF power measurement evaluation boards or compatible devices with powrms IIO interface
  • USB or Ethernet connection to the RF power measurement device
  • Stable IIO context for device communication

Dependencies

  • Data Logger Plugin: Must be installed and enabled (the RF Power Meter plugin extends Data Logger functionality)
  • IIO drivers for the target hardware

How to Use

  1. Ensure the Data Logger plugin is installed and enabled.
  2. Connect your RF power measurement device to your computer.
  3. Open Scopy and the plugin will automatically detect compatible devices.
  4. The RF Power Meter interface will transform the Data Logger display to show power and voltage measurements.
  5. Monitor real-time forward/reverse power readings and temperature-compensated voltage values.
