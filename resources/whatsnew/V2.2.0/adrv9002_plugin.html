ADRV9002 (Jupiter) Plugin

Scopy now offers dedicated support for the ADRV9002 (Jupiter) dual-channel RF transceiver.

The new ADRV9002 (Jupiter) Tool enables you to integrate and control ADRV9002-based devices within Scopy. It provides comprehensive features including device configuration, dual-channel RX/TX control, tracking calibrations, and an integrated profile generator.

Key Features

  • Device Global Settings: System-wide configuration and status monitoring including profile management and temperature monitoring
  • Dual RX/TX Channels: Independent control of receive and transmit signal paths with advanced gain control and calibration options
  • Profile Generator: Integrated tool for creating and managing ADRV9002 profiles with support for external TES integration
  • Tracking Calibrations: Real-time calibration controls for optimal RF performance

Profile Generator

The integrated Profile Generator provides a user-friendly interface for creating ADRV9002 device profiles. This feature requires the external libadrv9002-iio library and offers:

  • Preset profile templates
  • Channel configuration for RX1/RX2, TX1/TX2, and ORX channels
  • Real-time JSON configuration preview
  • Direct device loading and file export capabilities

External Tool Requirement

Note: The Profile Generator functionality requires the libadrv9002-iio library to be installed. Without this external tool, the Profile Generator will be disabled. The library is available at: https://analogdevicesinc.github.io/libadrv9002-iio

How to Use

  1. Connect your ADRV9002 device to your computer.
  2. Ensure the ADRV9002 IIO drivers are properly installed.
  3. Open Scopy and select the ADRV9002 tool from the available instruments.
  4. (Optional) Install the libadrv9002-iio library to enable Profile Generator functionality.

For more detailed information, please visit the official documentation.
