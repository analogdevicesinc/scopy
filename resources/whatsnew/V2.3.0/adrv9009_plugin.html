ADRV9009 Plugin

Scopy now offers comprehensive support for the ADRV9009 dual-channel RF transceiver, providing both basic and advanced configuration capabilities.

The new ADRV9009 Tool enables you to integrate and control ADRV9009-based devices within Scopy. It provides comprehensive features including device configuration, dual-channel TX/RX/ORX control, profile management, calibration controls, and FPGA-based signal processing.

Key Features

  • Comprehensive Dual-Channel Control: Independent TX/RX channel management with advanced gain control and signal path configuration
  • Multi-Device Support: Automatic device detection with MCS synchronization capabilities for multi-device setups
  • Advanced Configuration Tool: Dedicated advanced tool with 15 specialized navigation tabs for in-depth device control
  • Profile Management: Device profile configuration and calibration management with comprehensive tracking controls
  • FPGA Integration: FPGA-based signal processing capabilities with customizable settings
  • Observation RX (ORX): Dedicated observation receiver path for advanced RF monitoring

ADRV9009 Basic Tool

The main ADRV9009 plugin provides essential device control features including:

  • Global device settings and configuration
  • TX/RX chain control with gain and attenuation settings
  • Observation RX path management
  • Basic FPGA settings and status monitoring
  • Real-time device status and temperature monitoring

ADRV9009 Advanced Tool

The advanced tool provides comprehensive device control through 15 specialized navigation tabs covering all aspects of ADRV9009 configuration:

  • Detailed AGC (Automatic Gain Control) settings
  • Comprehensive calibration and BIST controls
  • Advanced clock and timing configuration
  • JESD204 interface settings and monitoring
  • GPIO and external control interfaces
  • Power management and optimization

Multi-Device Support

The ADRV9009 plugin includes sophisticated multi-device capabilities:

  • Automatic device detection and enumeration
  • MCS (Multi-Chip Synchronization) support for synchronized operation
  • Coordinated profile management across multiple devices
  • Unified control interface for complex RF systems

How to Use

  1. Connect your ADRV9009 device to your computer.
  2. Ensure the ADRV9009 IIO drivers are properly installed and the device is recognized.
  3. Open Scopy and select the ADRV9009 tool from the available instruments.
  4. Use the basic tool for essential configuration or access the advanced tool for comprehensive control.
  5. For multi-device setups, ensure all devices are properly connected and detected before configuring synchronization.

For more detailed information, please visit the official documentation.
