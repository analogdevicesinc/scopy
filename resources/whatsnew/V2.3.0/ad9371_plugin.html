AD9371 / AD9375 Plugin

Scopy now offers dedicated support for the AD9371 and AD9375 wideband RF transceivers.

The new AD9371 Tool enables you to integrate and control AD9371 and AD9375-based devices within Scopy. It provides real-time configuration of the full signal chain — receive, transmit, and observation paths — with live read-back from hardware on every change.

Key Features

  • Device Global Settings: Configure the Enable State Machine (ENSM) mode, enable or disable RX/TX calibrations, and load device profiles for specific sample rate and bandwidth combinations.
  • Receive Chain: Tune the RX LO frequency from 300 MHz to 6 GHz, set hardware gain in Manual Gain Control (MGC) mode, monitor RSSI, and select from Manual, Hybrid, or Automatic gain control modes.
  • Transmit Chain: Set TX LO frequency and per-channel attenuation (0 to −41.95 dB in 0.05 dB steps) for TX1 and TX2, with continuous QEC and LOL tracking options.
  • Observation / Sniffer Chain: Control the observation receiver LO frequency, hardware gain, gain control mode, and RF port selection for TX calibration monitoring.
  • Automatic AD9375 Detection: The plugin detects AD9375 hardware automatically and enables its exclusive Digital Pre-Distortion (DPD), Closed-Loop Gain Control (CLGC), and VSWR detection features on the transmit path.
  • Built-In Self-Test (BIST): Verify the device data path using tone injection, PRBS pattern generation, and digital loopback — all applied immediately without requiring a device reinitialize.
  • Advanced Tool: A 13-tab low-level configuration interface covering clock settings, TX/RX/ORx data path profiles, AGC tuning, JESD204B framer and deframer, GPIO, AUX DAC, and initialization calibrations.

How to Use

  1. Connect your AD9371 or AD9375 evaluation board to your computer.
  2. Ensure the AD9371 IIO drivers are properly installed.
  3. Open Scopy and connect to the device URI (e.g., ip:192.168.2.1).
  4. Select the AD9371 or ADRV9375 tool from the available instruments.

For more detailed information, please visit the official documentation.
