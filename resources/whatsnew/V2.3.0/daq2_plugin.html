DAQ2 Plugin

Scopy now offers dedicated support for the AD-FMCDAQ2-EBZ evaluation board.

The new DAQ2 Tool enables you to integrate and control DAQ2-based devices within Scopy. It provides ADC monitoring and test mode configuration for the AD9680, along with DAC data clock monitoring for the AD9144.

Key Features

  • ADC Monitoring: Displays the AD9680 sampling frequency read directly from the hardware.
  • Test Mode Control: Configures each ADC channel (Ch0 and Ch1) independently into built-in test modes including PN9, PN23, checkerboard, and more for digital interface validation.
  • DAC Monitoring: Displays the AD9144 data clock frequency read directly from the hardware.

How to Use

  1. Connect your AD-FMCDAQ2-EBZ device to a supported FPGA carrier board.
  2. Ensure the IIO drivers for the AD9680 and AD9144 are properly installed and the IIO context is accessible via network (IP) or USB.
  3. Open Scopy and select the DAQ2 tool from the available instruments.

For more detailed information, please visit the official documentation.
