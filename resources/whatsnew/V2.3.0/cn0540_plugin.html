CN0540 Plugin

Scopy now offers dedicated support for the CN0540 Precision Measurement System.

The new CN0540 Tool enables you to control and monitor CN0540-based precision measurement hardware directly within Scopy. It provides GPIO power management, FDA driver configuration, sensor voltage calibration, and real-time analog voltage monitoring.

Key Features

  • Power Control: Manage device power state via GPIO, including SW_FF fault flag detection, shutdown control, and constant current output.
  • ADC Driver Settings: Enable or disable the Fully-Differential Amplifier (FDA) and select between FULL POWER and LOW POWER operating modes.
  • Sensor Calibration: Read ADC input voltage, set the DAC shift voltage, compute sensor voltage, and run an automated iterative calibration routine.
  • Voltage Monitor: Display six analog pin voltages (Vin+, Vgpio2, Vgpio3, Vcom, Vfda+, Vfda-) refreshed automatically every second.

How to Use

  1. Connect your CN0540 board to your computer.
  2. Ensure the IIO drivers for ad7768-1, ltc2606, and one-bit-adc-dac are properly installed.
  3. Open Scopy and select the CN0540 tool from the available instruments.
  4. Use the Refresh button to read all hardware values, or interact with each collapsible section individually.

For more detailed information, please visit the official documentation.
