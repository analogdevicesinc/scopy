FMCOMMS11 Plugin

Scopy now offers dedicated support for the AD-FMCOMMS11-EBZ evaluation board.

The new FMCOMMS11 Tool enables you to integrate and control AD-FMCOMMS11-EBZ-based devices within Scopy. It provides a unified interface for configuring the high-speed RF DAC (AD9162), high-speed ADC (AD9625), input step attenuator (HMC1119), and output variable gain amplifier (ADL5240) through the IIO framework.

Key Features

  • ADC Control: Configure test mode and input scale for the AD9625 high-speed ADC, with live sampling frequency readback.
  • Input Attenuation: Adjust the HMC1119 step attenuator gain from 0 to −31.75 dB in 0.25 dB steps.
  • DAC Control: Set the NCO frequency and enable the FIR85 digital filter for the AD9162 RF DAC, with live sampling frequency readback.
  • Output VGA: Adjust the ADL5240 variable gain amplifier from −11.5 to +20 dB in 0.5 dB steps.
  • DDS Support: Direct digital synthesis for the DAC is handled by the generic DAC plugin working alongside this plugin.

How to Use

  1. Connect your AD-FMCOMMS11-EBZ device to your computer.
  2. Ensure the IIO drivers for the AD9625, AD9162, HMC1119, and ADL5240 are properly installed.
  3. Open Scopy and select the FMCOMMS11 tool from the available instruments.

For more detailed information, please visit the official documentation.
