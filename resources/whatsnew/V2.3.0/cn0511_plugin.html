CN0511 Plugin

Scopy now offers dedicated support for the CN0511 wideband synthesizer board.

The new CN0511 Tool enables you to integrate and control CN0511-based devices within Scopy. It communicates with the AD9166 DAC and the AD9166-AMP amplifier via the IIO subsystem, providing precise control over output frequency, amplitude, and amplifier enable state.

Key Features

  • Single Tone Mode: Configure the output frequency (0–10000 MHz) and amplitude (-91 to 0 dBm) of the AD9166 DAC channel
  • DAC Amplifier Control: Enable or disable the AD9166-AMP output amplifier stage with a single checkbox
  • Calibration Apply: Write frequency, amplitude, and IOFS calibration coefficients to the device atomically with the Enter button
  • Device Readback: Refresh button reads the current NCO frequency back from the hardware and updates the displayed value
  • Auto-Initialization: On connection, the AD9166 is initialized with the FIR85 interpolation filter and a 6 GHz DAC sampling frequency

How to Use

  1. Connect your CN0511 device to your computer.
  2. Ensure the AD9166 IIO drivers are properly installed.
  3. Open Scopy and select the CN0511 tool from the available instruments.
  4. Set the desired output frequency and amplitude, then click Enter to apply settings to the hardware.
  5. Use the Refresh button in the top-right corner to read back the current frequency from the device.

For more detailed information, please visit the official documentation.
