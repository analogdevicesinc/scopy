AD6676 Plugin

Scopy now offers dedicated support for the AD6676 wideband IF receiver subsystem.

The new AD6676 Tool enables you to integrate and control AD6676-based devices within Scopy. It provides full configuration of the ADC frequency, bandwidth, receive chain parameters, shuffler, and test mode settings.

Key Features

  • ADC Settings: Configure the ADC sampling frequency in the 2925–3200 MHz range
  • Bandwidth Settings: Set the IF bandwidth with automatic range enforcement (0.5%–5% of ADC frequency) and configurable low, high, and IF margins
  • Receive Settings: Control intermediate frequency, hardware gain, and scale (dBFS) for the receive chain
  • Shuffler Settings: Select shuffler control mode and threshold via dropdown menus
  • Test Settings: Apply built-in ADC test patterns for verification and debug

How to Use

  1. Connect your AD6676 device to your computer.
  2. Ensure the AD6676 IIO drivers are properly installed.
  3. Open Scopy and select the AD6676 tool from the available instruments.

For more detailed information, please visit the official documentation.
