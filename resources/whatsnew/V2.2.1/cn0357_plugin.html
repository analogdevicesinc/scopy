CN0357 Plugin

Scopy now offers dedicated support for the CN0357 Electrochemical Gas Concentration Sensor.

The new CN0357 Tool enables real-time monitoring of gas concentration by reading the AD7790 16-bit ADC and computing parts-per-million (ppm) values based on configurable sensor parameters and feedback resistance.

Key Features

  • Real-Time PPM Calculation: Continuously computes gas concentration from ADC readings.
  • ADC Settings: Configurable sampling frequency from 16 Hz to 120 Hz.
  • Dual Feedback Modes: Supports digital potentiometer (rheostat) and fixed resistor configurations.
  • Rheostat Programming: Write RDAC values directly to the AD5270 digital potentiometer.
  • Sensor Calibration: Adjustable sensor sensitivity for accurate ppm readings.
  • System Measurements: Displays conversion voltage, supply voltage, feedback resistance, and conversion coefficients.

How to Use

  1. Connect your CN0357 device to your computer.
  2. Ensure the CN0357 IIO drivers are properly installed.
  3. Open Scopy and select the CN0357 tool from the available instruments.

For more detailed information, please visit the official documentation.
