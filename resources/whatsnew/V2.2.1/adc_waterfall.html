ADC Waterfall Plot

The ADC Frequency instrument now includes a Waterfall Plot that displays spectrum evolution over time. Each FFT frame is added as a new row, with frequency on the X axis, time on the Y axis, and color encoding magnitude in dBFS.

Key Features

  • Time-Frequency View: Spot intermittent signals, drift, and transients.
  • Configurable History: 50 to 2000 rows (default 200).
  • Synchronized Intensity: Color scale follows the FFT Y-axis range.

Enable it from the plot settings menu under WATERFALL PLOT. See the official documentation for details.
